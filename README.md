# School Random Program

학교 자리 배치를 빠르게 만들고 관리하는 웹 프로그램입니다.

## 주요 기능
- 행/열(2~10) 설정으로 자리판 생성
- 자리 클릭으로 학생 이름 직접 입력
- `TXT 불러오기`로 이름 일괄 입력(줄바꿈/쉼표 구분)
- `랜덤 배치하기`로 이름 무작위 재배치
- 자리별 `고정`, `금지`, `X(비우기)` 제어
- `전체 지우기`로 초기화
- `뒤로가기/앞으로가기` 및 `Ctrl+Z`, `Ctrl+Y` 지원
- 자리 배치 상태 `저장(.json)` / `불러오기(.json)`
- 자리표를 `이미지(.png)`로 저장

## 사용 방법
1. 행/열 값을 정해 자리 수를 맞춥니다.
2. 자리 클릭 또는 `TXT 불러오기`로 이름을 넣습니다.
3. 필요한 자리는 `고정`하거나 `금지`로 설정합니다.
4. `랜덤 배치하기`를 눌러 배치합니다.
5. 결과를 `저장`, `불러오기`, `이미지 저장`으로 관리합니다.

## 파일 구성
- `index.html`: 화면 구조
- `style.css`: UI 스타일
- `script.js`: 자리 배치 로직, 파일 입출력, 랜덤/히스토리 기능
